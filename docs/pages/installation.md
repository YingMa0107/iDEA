`iDEA` is implemented as an
R package, `r BiocStyle::Githubpkg('xzhoulab', 'iDEA')`, which can be
installed from GitHub by:

```{r, eval = FALSE}
# install devtools if necessary
install.packages('devtools')

# install iDEA
devtools::install_github('xzhoulab/iDEA')

# load
library(iDEA)
```
